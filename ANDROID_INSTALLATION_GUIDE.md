# FieldCheck Android Installation Guide

## Quick Installation (5 minutes)

### What You Need
- Android phone (Android 8.0 or higher)
- USB cable (for file transfer) OR cloud storage/email
- `app-release.apk` file (56 MB)

---

## Step 1: Get the APK File

The APK is located at:
```

```

## Step 2: Allow Installation from Unknown Sources

1. Open **Settings** on your phone
2. Go to **Security** or **Safety**
3. Find **Unknown sources** or **Install unknown apps**
4. Toggle it **ON**

> For Android 10+: Settings â†’ Apps â†’ Special app access â†’ Install unknown apps â†’ Enable for your file manager

---

## Step 3: Install the APK

1. Open **Files** or **File Manager** app
2. Navigate to **Downloads** folder
3. Find **app-release.apk**
4. Tap on it
5. Tap **Install**
6. Wait for installation (about 1-2 minutes)
7. Tap **Open** when done

---

## Step 4: Grant Permissions

On first launch, the app will request:
- **Location:** Required for GPS tracking - Tap **Allow**
- **Camera:** For photo capture - Tap **Allow**
- **Storage:** For file access - Tap **Allow**

---

## Step 5: First Login

1. Enter your credentials (admin or employee account)
2. App will connect to backend server
3. You're ready to use FieldCheck!
