================================================================================
                         KEY ALGORITHMS
================================================================================

HAVERSINE FORMULA (Circular Geofence Validation):
  d = 2R × arcsin(√(sin²((lat2-lat1)/2) + cos(lat1) × cos(lat2) × sin²((lon2-lon1)/2)))
  
  Where:
  - d = distance between points
  - R = Earth's radius (6,371 km)
  - lat1, lon1 = employee's coordinates
  - lat2, lon2 = geofence center coordinates
  
  If d ≤ geofence radius: Employee is within geofence

RAY CASTING ALGORITHM (Polygon Geofence Validation - Future):
  1. Draw ray from point to infinity
  2. Count polygon edge crossings
  3. Odd count = inside, Even count = outside
  
  (Planned for future polygon geofence support)

================================================================================
                        HOW TO USE DIAGRAMS
================================================================================

ONLINE VIEWING (Easiest):
  1. Visit: https://www.plantuml.com/plantuml/uml/
  2. Copy content from any .puml file
  3. Paste into editor
  4. Click "Submit"
  5. View rendered diagram

VS CODE EXTENSION:
  1. Install "PlantUML" extension by jebbs
  2. Open any .puml file
  3. Press Alt+D to preview
  4. Export as PNG/SVG/PDF

COMMAND LINE:
  1. Install: npm install -g plantuml
  2. Generate PNG: plantuml 01_USE_CASE_DIAGRAM.puml
  3. Generate SVG: plantuml -tsvg 01_USE_CASE_DIAGRAM.puml

GITHUB:
  - GitHub automatically renders PlantUML diagrams
  - Include in markdown files with: ![Diagram](filename.puml)
