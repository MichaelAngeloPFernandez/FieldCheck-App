FIGURE 3.6: INPUT-PROCESS-OUTPUT (IPO) MODEL
================================================================================

```
┌──────────────────────────────────────────────────────────────────┐
│                    INPUT-PROCESS-OUTPUT MODEL                    │
└──────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│                          INPUT                                 │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  ┌──────────────────────────────────────────────────────────┐ │
│  │ 1. GPS Location Data                                     │ │
│  │    - Latitude, Longitude, Accuracy, Timestamp            │ │
│  │                                                          │ │
│  │ 2. Geofence Boundaries                                   │ │
│  │    - Center point & radius (circular)                    │ │
│  │    - Polygon coordinates                                 │ │
│  │                                                          │ │
│  │ 3. User Credentials                                      │ │
│  │    - Username, Password, JWT Token                       │ │
│  │                                                          │ │
│  │ 4. Task Data                                             │ │
│  │    - Task ID, Description, Assigned Employee, Due Date  │ │
│  │                                                          │ │
│  │ 5. Report Filters                                        │ │
│  │    - Date Range, Employee ID, Geofence ID               │ │
│  │                                                          │ │
│  └──────────────────────────────────────────────────────────┘ │
│                                                                │
└────────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌────────────────────────────────────────────────────────────────┐
│                       PROCESS                                  │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  ┌──────────────────────────────────────────────────────────┐ │
│  │ 1. Validate User Authentication                          │ │
│  │    - Verify JWT token                                    │ │
│  │    - Check user role (admin/employee)                    │ │
│  │                                                          │ │
│  │ 2. Validate Location Against Geofence                    │ │
│  │    - Use Haversine formula (circular)                    │ │
│  │    - Use Ray Casting algorithm (polygon)                 │ │
│  │    - Determine if inside/outside boundary                │ │
│  │                                                          │ │
│  │ 3. Process Attendance Record                             │ │
│  │    - Create attendance entry                             │ │
│  │    - Store timestamp and location                        │ │
│  │    - Mark as synced/unsynced                             │ │
│  │                                                          │ │
│  │ 4. Manage Tasks                                          │ │
│  │    - Assign tasks to employees                           │ │
│  │    - Update task status                                  │ │
│  │    - Track completion                                    │ │
│  │                                                          │ │
│  │ 5. Generate Reports                                      │ │
│  │    - Filter attendance data                              │ │
│  │    - Calculate statistics                                │ │
│  │    - Format for export (PDF/Excel)                       │ │
│  │                                                          │ │
│  │ 6. Broadcast Real-Time Updates                           │ │
│  │    - Send Socket.io events                               │ │
│  │    - Update connected clients                            │ │
│  │                                                          │ │
│  └──────────────────────────────────────────────────────────┘ │
│                                                                │
└────────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌────────────────────────────────────────────────────────────────┐
│                       OUTPUT                                   │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  ┌──────────────────────────────────────────────────────────┐ │
│  │ 1. Attendance Record                                     │ │
│  │    - Success/Failure message                             │ │
│  │    - Check-in/out timestamp                              │ │
│  │    - Location coordinates                                │ │
│  │                                                          │ │
│  │ 2. Real-Time Notification                                │ │
│  │    - Socket.io event to admin dashboard                  │ │
│  │    - Updated employee location                           │ │
│  │    - Attendance status change                            │ │
│  │                                                          │ │
│  │ 3. Task Update                                           │ │
│  │    - Task assignment confirmation                        │ │
│  │    - Task status update                                  │ │
│  │    - Completion notification                             │ │
│  │                                                          │ │
│  │ 4. Report Export                                         │ │
│  │    - PDF file generated                                  │ │
│  │    - Excel file generated                                │ │
│  │    - File saved to device/server                         │ │
│  │                                                          │ │
│  │ 5. Error Messages                                        │ │
│  │    - "Outside geofence boundary"                         │ │
│  │    - "Invalid credentials"                               │ │
│  │    - "Offline - data will sync later"                    │ │
│  │                                                          │ │
│  │ 6. Database Update                                       │ │
│  │    - Attendance record stored                            │ │
│  │    - Task status updated                                 │ │
│  │    - Report generated                                    │ │
│  │                                                          │ │
│  └──────────────────────────────────────────────────────────┘ │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```
